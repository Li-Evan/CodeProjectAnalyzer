# Overview
这是一个辅助阅读项目代码的工具，它会给出一个项目整体架构分析，以及每个项目文件具体的功能和分析
---
# Quick Strat
1. 在main.py处传入想要分析的项目的路径
2. 运行后在output文件夹里面找到分析的json文件
3. 运行frontend.py，可以上传第2步得到的json文件，得到更好的阅读体验
